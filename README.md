## Linked List Topics Covered in Detail

### Insertion (`04_linklist_insertion.c`)

- Insert at beginning (`insertAtFirst`) - `O(1)`
- Insert at specific index (`insertAtIndex`) - `O(n)`
- Insert at end (`insertAtEnd`) - `O(n)`
- Insert after a given node (`insertAfterNode`) - `O(1)`

### Deletion (`05_linklist_deletion.c`)

- Delete first node (`deleteFirst`)
- Delete node at specific index (`deleteAtIndex`)
- Delete last node (`deleteAtLast`)
- Delete node by value (implemented in last delete function)
